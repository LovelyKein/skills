## 项目简介

**chigua** 是一个自包含的 AI Agent 技能包。从吐槽中自动构建人际关系图谱，并提供个性化社交应对建议。

角色定位：

- **倾听者** — 先共情，再分析
- **关系顾问** — 基于图谱历史数据，给出个性化建议
- **魏征式镜子** — 敢于指出用户的自相矛盾和行为盲区

核心能力：

- 从对话中自动提取人物、事件、关系变化
- 构建以"我"为中心的人际关系图谱
- 基于历史数据给出个性化社交应对建议
- 支持关系网络可视化（Mermaid 图）

## 快速使用

触发关键词：吐槽人际关系、分析与人相处的困境、理清某人图谱、"该怎么回/怎么做"

| 场景         | 输入示例                 |
| ------------ | ------------------------ |
| 吐槽         | "今天我同事又甩锅给我了" |
| 求助分析     | "我和XX的关系怎么样"     |
| 全局关系     | "我的关系网什么样的"     |
| 可视化       | "画个关系图"             |
| 自我模式分析 | "我是不是总在重复XX"     |
