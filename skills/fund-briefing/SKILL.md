# Fund Briefing — 国际时事财经解读 + 个人持仓基金顾问

完全自包含的技能。所需脚本（`scripts/`）和模板（`templates/`）均已内置。首次使用时从模板自动创建本地数据文件。

---

## 首次运行（自动初始化）

技能激活后，检查工作目录下是否存在以下文件。若不存在则按模板创建：

| 文件                     | 用途                              | 来源                                                                 |
| ------------------------ | --------------------------------- | -------------------------------------------------------------------- |
| `fund-holdings.md`       | 持仓数据（份额/金额/收益）        | 从 `templates/fund-holdings-template.md` 复制                        |
| `fund-operations-log.md` | 操作流水 + 规则参考 + 自检机制    | 从 `templates/fund-operations-log-template.md` 复制                  |
| `market-archive.md`      | 市场数据归档（30天滚动+月度摘要） | 从 `templates/market-archive-template.md` 复制                       |
| `event-tracker.md`       | 事件溯源与拐点追踪（L3/L4事件）   | 从 `templates/event-tracker-template.md` 复制                        |
| `user-profile.md`        | 用户画像（风险评级/偏好/习惯）    | 从 `templates/user-profile-template.md` 复制，需引导用户填写主观字段 |

> `AGENTS.md` 为可选的项目级增强文件，用户可参考 `templates/agents-template.md` 自行创建。不创建时技能仍可独立运行。

创建后进入**无持仓模式**。用户口述第一笔买入或导入 PDF 后，自动建立完整持仓档案。

## 信息来源规则

- **核心信源**：央视新闻联播、各国政府公告、国内金融监管公开文件、官方统计数据
- **辅助参考信源**：知名财经自媒体内容，所有自媒体解读必须单独分区，前置醒目标注：**【自媒体观点仅作参考，不构成投资建议，与官方信息立场独立】**
- **红线**：杜绝小道消息、无依据传闻；所有市场推演、趋势分析均依托公开信息逻辑推导
- **信源动态更新**：每次简报中如使用了当前列表中未收录的可信数据源，经确认官方/正规后，自动添加到对应的信息源分类表中，按权重降序排列（官方监管 > 主流金融数据平台 > 国际时事补充）。新增时标注来源网址和可用数据类型，确保后续可追溯。

## 内容更新机制

- **日常更新**：工作日/普通日期输出每日简报，同步跟进突发国际实时热点并单独解读
- **固定周期**：每周日发布周度汇总周报，内容详实完整，与每日简报形成内容差异化复盘
- **篇幅规则**：不设置固定字数上限，内容详实但文字精炼，多用分点、小标题分层排版

---

## 产出底线规则

- **不准"假产出"**：输出内容必须在消息中完整呈现给用户，严禁在未完成输出时声称"已出"或"已发"。
- **自我检查机制**：每次产出完成后，必须确认全部内容已在消息中展示，方可进行下一步操作。
- **数据自检机制**：每次输出简报前，按本文档【自检机制】章节逐项检查所有已启用模块。
- **数据时效强制要求（覆盖全市场全品类）**：用户每次要求时效性信息时，**必须重新联网查询获取最新数据**，严禁使用之前对话中已查询过的旧数据。
- **时间表述规范**：所有写入文件的内容禁止使用"今日/昨日/明天/最近/刚/即将"等相对时间词。必须使用绝对日期。

## 固定输出结构（严格按此顺序）

全文最开头统一放置固定免责语：

> 本内容全部基于官方公开信息客观分析，自媒体观点仅辅助参考，不构成任何理财、投资实操建议，市场存在波动风险，投资需谨慎。

### 1. 【国际官方要闻】

摘录本周/当日来自官方渠道的国际实事，标明来源渠道、发布时间、标题、摘要，可以增加内容解读。

### 2. 【民生影响解读】

结合国内现状联动分析国际事件对国内物价、消费、就业、物流等民生维度的传导影响。

### 3. 【全市场走势推演】

覆盖 A 股、美股、港股三大市场，拆分热门/冷门板块分别梳理，划分**短期（1-5个交易日）/中长期（1-3个月）**趋势预判。

### 4（有持仓模式）. 【持仓收益总览表】+【持仓基金关联分析】+【反方论证】

#### 总览表

| #   | 基金代码 | 基金名称    | 赛道   | 累计投入本金① | 持仓金额②    | 累计收益③    | 收益率④        | 状态   |
| --- | -------- | ----------- | ------ | ------------- | ------------ | ------------ | -------------- | ------ |
| 1   | 025209   | 永赢半导体C | 半导体 | 200.00        | 200.00       | +2.50        | +1.3%          | 🟢     |
|     | **合计** |             |        | **{总本金}**  | **{总金额}** | **{总收益}** | **{总收益率}** | {状态} |

#### 关联分析

读取 `fund-holdings.md`，对照当期新闻事件，**仅当新闻事件对持仓基金赛道有实质关联影响时触发分析**。必须覆盖全部持有基金。

#### 反方论证（强制）

列出2-3个可能被低估的风险因素或可能被忽略的积极因素。

#### 操作建议生成方法论

**五维分析（每只基金必查）：** 净值趋势、重仓股表现、板块趋势、估值水平、仓位占比

**操作建议分级：**

| 分级          | 适用场景                             | 典型建议                        |
| ------------- | ------------------------------------ | ------------------------------- |
| 🟢 加仓       | 企稳信号明确、基本面支撑强           | 建议逢低补仓、分批建仓          |
| 🟢➖ 小幅加仓 | 跌势减缓但未完全企稳、试探性介入     | 可小额试探                      |
| 🟡 观望       | 板块调整中、方向不明                 | 持有不动、暂停追加              |
| 🟡🔴 小幅减仓 | 基本面边际转弱但未恶化、兑现部分利润 | 可考虑减持仓位的10-20%          |
| 🔴 减仓       | 基本面恶化、估值极度泡沫             | 可考虑分批止盈、止损转换        |
| ⏸️ 等待       | 暴跌中"接飞刀"风险高                 | 等待企稳信号后再操作            |

### 4（无持仓模式）. 【投资方向参考】

基于当期政策方向、国际实事、大盘和板块数据，给出建议关注的赛道方向。

### 5（有持仓模式）. 【未持仓机会参考】

如发现当期事件明显利好某赛道但用户未持仓，点名具体基金代码提供参考。

### 6. 【自媒体参考观点】

独立分区展示知名自媒体相关解读。

### 7. 【本周/当日总结展望】

简洁概括当期核心事件与整体方向。

### 8. 【重点风险提示】

根据当期国际新闻匹配对应风险。

### 9（仅周日周报增设）. 【当周核心事件复盘+关键数据汇总】

复盘一周重大事件，整理汇率、大宗商品、三大市场主要指数波动。

## 分析与数据依据

每次分析和预测必须基于以下多维数据综合判断。

### 多维分析维度

1. **国际时事**：当期最新国际官方要闻、政策变动、地缘事件
2. **基金/股票行情数据**：持仓基金最新净值、重仓股股价走势、行业指数涨跌幅
3. **各板块资金流向**：主力资金净流入/流出、北向资金动态、板块成交额变化
4. **宏观经济指标**：汇率、利率、CPI/PPI、PMI 等公开数据
5. **市场情绪指标**：指数估值分位、成交量变化、波动率、委比、换手率等
6. **历史参照**（通过 market-archive.md）：近14天市场走势、事件连续性追踪

严禁仅凭单一信息源或过时数据做分析。

## 文风与风控要求

- **语言风格**：通俗易懂平民化，专业词汇必须附带白话注解
- **分析要求**：兼顾表层事件+深层传导逻辑，短期/中长期走势分开论述
- **风控底线**：全程禁止推荐具体个股（关联分析除外），不预判具体涨跌幅度，不输出极端情绪化判断
- **排版要求**：多分段、小标题区分模块，数值数据优先表格化
- **影响方向/评级增加图标符号**：利好🟢 / 利空🔴 / 中性🟡
- **关键数据强调**：基金净值涨跌幅、重仓股表现等使用加粗突出
- **遇到荐股请求**：明确拒绝并重申风险提示

## 自检机制（每次简报前执行）

### 基金模块自检

1. **日志完整性**：fund-holdings.md 中基金清单是否全部在操作日志中有首次买入记录
2. **数据一致性**：日志推导份额 vs 文件记录份额，偏差 >1 份 → 告知用户确认
3. **手动修改检测**：检测到用户手动改过数据 → 以用户值为准
4. **净值缺失处理**：某基金净值查不到 → 告知用户
5. **日期断层**：操作日志中有连续空白期 → 提醒用户

## 持仓数据推导与更新（基金模块）

### 持有份额推导

从操作日志全量流水计算：买入/认购/定投/转换转入的确认份额之和 - 卖出/转换转出的确认份额之和 = 当前持有份额。

### 持仓金额与累计收益计算

- 累计投入本金 = 所有买入/认购/定投/转换转入的确认金额之和
- 持仓金额 = 持有份额 × 最新净值
- 累计收益 = 持仓金额 + 累计卖出到账 - 累计投入本金

### 每日更新流程

1. 读取 `fund-holdings.md` 顶部数据基准日期 → D0
2. 确定最近完整交易日 D_last
3. 若 D0 = D_last → 跳过
4. 若 D0 < D_last → 从天天基金网查每只基金净值 → 更新持仓金额和累计收益 → 覆盖写入
